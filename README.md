# CALUMI.Motion by [Calaverah](https://www.github.com/Calaverah)

CALUMI Motion provides a user interface to view and edit Starfield Animation Behavior Graphs (.agx)

The application is currently in a functional MVP state, and will see improvements and changes overtime. It is heavily advised to review and validate your agx output!


## Features

- Import/Export .agx files!
- Custom save format with additional features such as group colors!
- View detailed information in a more user friendly, property sheet style, format!

## Related

This project was creating using: 
- [Qt6](https://github.com/qt/qtbase) by Qt (LGPLv3)
- [qtnodes](https://github.com/paceholder/nodeeditor) by Paceholder (BSD-3)
- [pugixml](https://github.com/zeux/pugixml) by Zeux (MIT)

Heavy modification/recreation of qtnodes turned what was once a dependency fork into a single project of completely modified and original elements. Ultimately, the fork was disconnected once the project code had become a monster of its own design. While not direct, this project is spiritually dependant on nodeeditor and owes an eternal gratitude for their support.
